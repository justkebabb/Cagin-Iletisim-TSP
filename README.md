# Çağın İletişim TSP
### Teknik Servis Programı

Telefon teknik servisleri için geliştirilmiş ücretsiz, açık kaynaklı masaüstü yönetim programı.

## Özellikler
- Müşteri ve cihaz kaydı (zorunlu alan yok, hızlı kayıt)
- Durum takibi: Bekleyenler / Tamirde / Teslim Edildi
- Canlı arama (müşteri adı, model, problem)
- Tarih bazlı kayıt gruplandırma
- Kar analizi sayfası (günlük)
- IMEI Türkiye kontrolü
- Otomatik ve manuel yedekleme sistemi
- Google Drive klasörüne yedekleme
- Gece / Gündüz modu
- EXE olarak çalışır, kurulum gerektirmez

## Kurulum (Python ile)
```bash
pip install -r requirements.txt
python main.py
```

## Hazır EXE
Releases bölümünden EXE dosyasını indirip direkt çalıştırabilirsiniz.
